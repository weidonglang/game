# Dice Roll Duel Mini

A minimal Python terminal game where you duel the computer by rolling two six-sided dice each round.

## How to Play

1. Run the game:

```bash
python dice_duel.py
```

2. Press Enter to roll your dice.
3. The computer rolls automatically.
4. Highest total wins the round.
5. After each round, choose whether to continue.

## Run Tests

```bash
python -m unittest discover dice-roll-duel-mini
```
