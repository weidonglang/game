# MATLAB Arcade Simulation Lab

A comprehensive collection of probability-based mini-game simulations and analysis tools written in MATLAB.

## Project Structure

```
matlab-arcade-simulation-lab/
├── README.md
├── run_all_demos.m      # Run all demo scripts
├── run_all_tests.m      # Run all unit tests
├── src/
│   ├── core/            # Core utility functions
│   ├── games/           # Game simulation modules
│   ├── strategies/      # AI strategy functions
│   ├── simulations/     # Tournament and batch simulation engines
│   ├── analytics/       # Statistical analysis functions
│   └── reports/         # Text report generators
└── tests/               # Unit tests
```

## Requirements

- MATLAB R2016b or later (or GNU Octave 4.2+)
- No external dependencies required

## Usage

Run all demos:
```matlab
run_all_demos
```

Run all tests:
```matlab
run_all_tests
```
