# Multi-Language Luck Lab

A small multi-language project exploring luck mini-games, probability simulation, and data analysis.

## Modules

| Module | Language | Description |
|--------|----------|-------------|
| `c-wheel/` | C | Lucky wheel terminal game |
| `cpp-score/` | C++ | Score statistics and leaderboard analyzer |
| `r-analysis/` | R | Game result statistics script |
| `matlab-sim/` | MATLAB | Probability simulation and expected value analysis |

## Requirements

- C99 or later (for C module)
- C++17 or later (for C++ module)
- R 4.0+ (for R module)
- MATLAB R2020a+ (for MATLAB module)
