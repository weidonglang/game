# C++ Luck Lab Score Analyzer

A C++ program that analyzes player scores from lucky games and prints a leaderboard.

## Features

- Total score calculation
- Average score calculation
- Highest and lowest score identification
- Sorted leaderboard display
- Win count statistics

## Files

| File | Description |
|------|-------------|
| `score_analyzer.cpp` | Score analyzer source code |

## Compilation

```bash
g++ -std=c++17 -Wall -Wextra -pedantic score_analyzer.cpp -o score_analyzer.exe
```

## Usage

```bash
./score_analyzer.exe
