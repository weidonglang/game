# MATLAB Luck Lab Simulation

A MATLAB simulation script that models a coin-based lucky game for multiple players, tracking scores, win rates, and streak patterns.

## Files

| File | Description |
|------|-------------|
| `simulate_luck_game.m` | MATLAB simulation script |

## Requirements

- MATLAB or GNU Octave

## Usage

In MATLAB command window:

```matlab
>> simulate_luck_game
```

Or from the terminal:

```bash
matlab -batch "simulate_luck_game"
```

With Octave:

```bash
octave --eval simulate_luck_game
